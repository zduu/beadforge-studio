# 拼豆图纸生成器

一个面向 Cloudflare Pages 的前端应用，用于把上传图片转换成单层拼豆图纸。当前版本使用 Bambu Lab PLA Basic 色卡进行近似颜色匹配，并支持颜色替换、实时预览和导出。

## 已实现

- 上传 PNG、JPG、WebP 图片。
- 默认生成 32x32 图纸，宽高输入不设置固定最大值。
- 默认 `contain` 适配，完整保留图片并允许留白。
- 使用 Bambu Lab PLA Basic 近似色卡匹配颜色。
- 支持限制最大颜色数，输入 `0` 表示不限制颜色数量。
- 支持指定背景色，背景格不参与拼豆数量统计、CSV 导出和图纸拼豆显示。
- 背景色按当前选中颜色全局排除；选红色时红色不显示，切换到黄色时红色恢复、黄色不显示，选择不排除时全部显示。
- 图片未覆盖到的空白区域会自动填入当前背景色。
- 重新生成图纸时会保留当前背景色设置。
- 支持平均色和中心点两种取样方式。
- 支持细节增强，对低分辨率图纸做网格级锐化、对比度和饱和度增强。
- 支持在原图上拖拽矩形框裁剪，方框外的图片会在重新生成时被去除。
- 支持在当前预览图纸上拖拽矩形框裁剪，裁剪后图纸会实际缩小到选区尺寸。
- 支持水平镜像后再生成图纸。
- 支持在预览区放大/缩小查看图纸，放大后可拖动查看局部。
- 低于约 48 格的图纸会自动加强轮廓和对比，改善 35x35 这类尺寸下的可读性。
- 宽高和颜色数量输入在失焦或按回车后校验，不会在输入过程中强制改值。
- 支持 32、48、64、80 常用尺寸快捷按钮。
- 支持点击颜色统计项后批量替换颜色。
- 支持选择画笔颜色并点击单个格子修改，画笔颜色使用中文名称显示。
- 支持画笔、橡皮、查看三种工具模式，默认进入查看模式。
- 支持回退编辑操作，覆盖画笔、橡皮、颜色替换和背景色变更。
- 橡皮会把格子擦成当前背景色；如果未设置背景色，则擦为空白格。
- 查看模式点击格子只显示坐标和颜色信息，不修改图纸。
- 支持导出 PNG、CSV、JSON。
- 支持导入 JSON 项目文件继续编辑。
- 支持上传 STL / 3MF 模型，按 2.6mm 格距和 3mm 层高生成基础多层图纸。
- 支持切换查看模型切片的每一层。
- 支持整体 3D 模型预览，可旋转查看多层叠加效果。
- 支持导出多层项目 JSON 和颜色映射 JSON。
- 桌面端左右设置栏独立滚动，中央预览区保持可见。
- 左侧 3D 模型设置默认折叠，避免干扰图片图纸设置。

## 3D 模型规划

当前已实现 STL / 3MF 文件的浏览器端基础切片。切片会把模型缩放到当前图纸宽高范围内，并使用单一默认 Bambu PLA Basic 颜色填充每一层。

- 平面格距：2.6mm
- 单层高度：3mm

当前限制：

- STL 通常不包含颜色，因此按用户选择的默认耗材颜色生成。
- 3MF 当前只读取几何，不解析 3MF 内置材质、纹理或 Bambu Studio 扩展颜色信息。
- 大模型会影响浏览器性能，建议先使用低面数模型和 96x96 以内的图纸尺寸。
- 整体 3D 预览会对超大模型做抽样显示，避免浏览器卡死。
- 虽然宽高没有固定最大值，浏览器端图片处理仍受内存限制；尺寸过大时应用会提示降低宽高后重试。

后续建议实现带颜色 3MF 的材质读取和 Bambu 色卡映射。

## 本地开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

构建产物输出到 `dist/`。

## Cloudflare Pages 部署

Cloudflare Pages 设置：

- Framework preset: `Vite`
- Build command: `npm run build`
- Build output directory: `dist`
- Node.js version: `24` 或 Cloudflare 当前支持的最新 LTS

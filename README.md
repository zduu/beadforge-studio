# 拼豆图纸生成器

一个面向 Cloudflare Pages 的前端应用，用于把图片或 STL / 3MF 模型转换成拼豆图纸。图片路径使用 Bambu Lab PLA Basic 色卡做近似匹配；3MF 模型会优先读取文件内的耗材颜色和 Bambu Studio 涂色信息，生成多层图纸、颜色统计和 3D 预览。模型解析、预览和切片在本机浏览器 Web Worker 中执行，不上传到服务器处理。

## 功能概览

### 图片生成

- 上传 PNG、JPG、WebP 图片并生成单层图纸。
- 默认生成 32x32 图纸，支持 32、48、64、80 快捷尺寸，也支持手动输入宽高。
- 支持 `contain`、`cover`、`stretch` 三种适配方式。
- 支持平均色和中心点两种取样方式。
- 支持限制最大颜色数，输入 `0` 表示不限制颜色数量。
- 支持细节增强，对低分辨率图纸做网格级锐化、对比度和饱和度增强。
- 支持原图裁剪、预览图裁剪、水平镜像和预览缩放拖动。
- 低于约 48 格的图纸会自动加强轮廓和对比，改善小尺寸图纸可读性。

### 编辑与导出

- 支持背景色排除，背景格不参与拼豆数量统计、CSV 导出和图纸拼豆显示。
- 图片未覆盖到的空白区域会自动填入当前背景色。
- 支持画笔、橡皮、查看三种工具模式，默认进入查看模式。
- 支持点击颜色统计项选择颜色，并批量替换为其他 PLA Basic 颜色。
- 支持回退编辑操作，覆盖画笔、橡皮、颜色替换和背景色变更。
- 支持顺时针旋转当前二维图纸。
- 支持导出 PNG、CSV、JSON。
- 支持导入 JSON 项目文件继续编辑。

### 3D 模型切片

- 支持上传 STL / 3MF 模型并生成多层图纸。
- 上传模型后先显示原始 3D 预览，不立即切片。
- 支持切片前绕 X/Y/Z 轴调整模型方向。
- 支持配置模型切片参数：层数、格距、层高和默认耗材。
- 支持上传后调整参数并重新切片。
- 模型解析和切片在本机浏览器 Worker 中执行，并支持取消本机处理任务。
- 支持切换查看每一层，当前层视图支持顺时针旋转。
- 支持整体 3D 模型预览，可旋转查看多层叠加效果。
- 支持导出多层项目 JSON 和颜色映射 JSON。
- 支持 Bambu Studio 3MF 的 `filament_colour`、整零件 `extruder` 和三角面 `paint_color`。
- 3MF 内部耗材颜色会保留为独立 `3MF Filament` 色号，避免浅色被强行合并到内置 PLA Basic 色卡。

## 3D 切片规则

### 默认尺寸

当“层数”为空或为 `0` 时，模型按当前图纸宽高和格距做默认缩放：

```text
scaleX = 图纸宽度 * 格距 / 模型原始宽度
scaleY = 图纸高度 * 格距 / 模型原始深度
默认缩放 = min(scaleX, scaleY)
默认层数 = ceil(模型原始高度 * 默认缩放 / 层高)
```

默认参数：

- 格距：`2.6mm`
- 层高：`3mm`
- 层数：空或 `0`，表示使用默认自然层数

### 指定层数

当“层数”大于 `0` 时，系统先计算默认大小，再按目标层数等比例缩放整个模型。也就是说，层数不是截断上限，而是控制模型整体缩放的目标高度。

### 颜色来源优先级

模型切片颜色按以下顺序决定：

1. Bambu Studio 3MF 三角面 `paint_color`
2. Bambu Studio 3MF 零件或对象 `extruder`
3. 上传面板里选择的默认耗材

STL 通常不包含颜色，因此会使用默认耗材。

## 当前限制

- 3MF `paint_color` 已覆盖当前样本中的 Bambu Studio 编码，但不同切片软件可能存在未覆盖的扩展格式。
- 纹理贴图型 3MF 暂未按 UV / bitmap 颜色切片。
- 本机浏览器切片对超大模型仍有性能压力，建议优先使用低面数模型和 96x96 以内图纸尺寸。
- 整体 3D 预览会对超大图纸抽样显示，保证可交互性，但不是完整渲染所有方块。
- 当前二维切片是模型几何与水平切面交线填充，复杂非流形、薄壁或自相交模型可能出现孔洞或局部颜色不稳定。

## 改进优化计划

### P0：稳定性与颜色兼容

- 建立 3MF 回归样本集，覆盖 `Flexi+Contadino.3mf`、草地牛以及更多 Bambu Studio 多色文件。
- 为 `paint_color`、`extruder`、原始耗材色保留增加自动化统计测试，防止多色文件退化成单色或少色。
- 补充更多 3MF 材质来源：base material、color group、对象材质属性和非 Bambu 扩展。

### P1：切片质量

- 增加模型方向控制，在切片前支持绕 X/Y/Z 轴旋转模型。
- 改进切片填充规则，减少复杂模型中的孔洞、错配和细小孤岛。
- 增加层预览诊断信息：实际层数、缩放比例、模型占用尺寸、每层颜色数。
- 支持按层隐藏背景、孤岛过滤和最小区域阈值。

### P2：性能与交互

- 继续完善 Web Worker 任务进度，细分解压、解析、归一化和逐层切片阶段。
- 完善取消任务后的状态恢复和错误诊断。
- 对大 3MF 做分块解析和更细的抽样预览策略。
- 优化整体 3D 预览的实例化数量和层级可视化。

### P3：导出与生产流程

- 增加按层导出 PNG / CSV 批量包。
- 增加颜色换料计划、层色统计和耗材槽映射导出。
- 支持项目文件保存模型切片参数，导入后可继续重新切片。
- 增加打印/拼装说明视图，方便按层制作和核对。

## 本地开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

构建产物输出到 `dist/`。

## Cloudflare Pages 部署

Cloudflare Pages 设置：

- Framework preset: `Vite`
- Build command: `npm run build`
- Build output directory: `dist`
- Node.js version: `24` 或 Cloudflare 当前支持的最新 LTS
